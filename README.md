# @candlefinance/page-control

UIPageControl wrapper for react native

## Installation

```sh
npm install @candlefinance/page-control
```

## Usage

```js
import { PageControlView } from "@candlefinance/page-control";

// ...

<PageControlView color="tomato" />
```
